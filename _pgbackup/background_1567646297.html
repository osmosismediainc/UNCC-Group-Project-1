The Journey To Our API Application

The Idea

Music is universal. Our team, team Beastmasters, decided we would make an API application to look up lyrics of music.

The APIs

AudD.io

A lot of developers use AudD to recognize music in their own applications (to identify what the song is playing somewhere). Also, they display the lyrics of the recognized songs or help their users find lyrics by text.

Visit This API
Elvish Translation - Quenya Translator

Translate from English to Elvish languages Sindarin or Quenya. Elvish is a collection of fictional languages spoken by an immortal species in middle earth in the movie Lord of the Rings. Sindarin and Quenya are the main Elvish languages. This API allows for conversion from English to Sindarin or Quenya.
